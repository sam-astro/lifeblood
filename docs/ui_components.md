# UI elements
headings look like this:

# heading 1
## heading 2
### heading 3
#### heading 4
##### heading 5
###### heading 6

body text looks like this. it can be **bold**, *italic*, or <s>strikethrough</s>.

code blocks look like this:
```
code line 1
code line 2
```

highlighting is also supported:
```js
// code with highlighting
if (thing == true) {
  object.method();
  return true;
}
```
